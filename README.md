## 🛠️ Core Features

* **Voice Module (TTS):** Integrated speech synthesis for audible system feedback.
* **Edge Processing:** All logic (math, encryption, analysis) happens locally in the browser.
* **Privacy First:** Zero data is transmitted to external servers.
* **Command-Line Interface:** A nostalgic yet powerful terminal UI for modern web users.
* **Persistence:** Built-in network access logger using `localStorage`.

---

## ⌨️ Command Registry

B.R.U.N.O. uses a strict `/` syntax. Below are the primary functional modules:

| Command | Description | Example |
| :--- | :--- | :--- |
| `/calc` | Evaluates complex mathematical expressions | `/calc (50 * 12) / 2` |
| `/pass` | Generates a high-entropy secure password | `/pass 16` |
| `/timer` | Sets a local countdown alert with voice feedback | `/timer 60` |
| `/encode` | Converts plain text into secure Base64 strings | `/encode MySecret` |
| `/roll` | Simulates a die roll (defaults to D6) | `/roll 20` |
| `/help` | Displays the full system documentation | `/help` |

## 📜 Technical Whitepaper
B.R.U.N.O. is built on an ultra-lightweight JavaScript kernel. By prioritizing "Basic" operations, we eliminate the overhead associated with large-scale LLMs, providing a tool that is as fast as the hardware it runs on.
